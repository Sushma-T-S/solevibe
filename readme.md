# Solevibe Project

**Description:**  
Solevibe is a full-stack e-commerce project with separate folders for client, admin, and server.  
It includes product listing, cart, order management, and admin panel features.

## Project Structure

- `client/` → Frontend (React.js)  
- `admin/` → Admin Panel (React.js)  
- `server/` → Backend (Node.js / Express)
